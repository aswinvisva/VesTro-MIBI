MIBI Microenvironment Identification documentation
===============================================================================

Python Toolbox for ML-based microenvironment identification from MIBI data.

Installation & Getting Started
==============================
.. highlight:: console

1. Fetch the source code

.. code-block:: console

   $ git clone https://github.com/aswinvisva/oliveria-lab-ml.git

2. Create a virtual environment

.. code-block:: console

   $ python3 -m venv env

3. Activate the virtual environment

.. code-block:: console

   $ source env/bin/activate

4. Install requirements:

.. code-block:: console

   $ pip install -r requirements.txt

5. Run Demo

.. code-block:: console

   $ python3 run.py
